# DocuMind

Intelligent Document Analysis and Structuring System

## Overview

DocuMind is a lightweight document analysis system that combines OCR and large language models for structured document understanding.

The system supports:
- PDF and image input
- OCR text extraction
- OCR quality evaluation
- Structure detection
- AI-generated summaries
- Follow-up question answering

## Technologies

- Python
- Flask
- Tesseract OCR
- Gemini API
- HTML / CSS / JavaScript
